# MilkDrop3 Omni-Workspace

[![Version](https://img.shields.io/badge/version-dynamic-blue)](VERSION.md)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

Welcome to the **MilkDrop3 Omni-Workspace**, a centralized manager and command center for the MilkDrop 3.0 ecosystem and a diverse set of AI-driven projects.

## 🌟 Vision
This repository serves as an **Omni-Workspace**—an autonomous, self-healing, and self-documenting software ecosystem where AI agents and human developers collaborate across 100+ nested repositories. For more details, see [VISION.md](VISION.md).

## 📂 Project Structure
Following the [Universal LLM Instructions](docs/UNIVERSAL_LLM_INSTRUCTIONS.md), the workspace is organized into several key areas:

- `docs/`: Workspace-level documentation and universal instructions.
- `scripts/`: Automation and maintenance scripts (e.g., `update_repos_v5.py`).
- `logs/`: Operation logs and handoff archives.
- `aios/`, `borg/`, `metamcp/`: AI Orchestration layers.
- `bobmani/`: Rhythm game engine suite.
- `fwber/`, `bobcoin/`: Full-stack applications.
