# Changelog - MilkDrop3 Omni-Workspace

All notable changes to this project will be documented in this file.

## [3.1.0] - 2026-06-24
### Added
- Synchronized missing submodules (`aios`, `bg`, `bobcoin`, `itgmania`, `okgame`).

### Added
- Modernized **Web Dashboard** with a responsive Dark Mode layout and component cards.
- Implemented **Omni-Search** engine for ecosystem-wide grep discovery.
- Automated the **Release Cycle** (versioning, docs, handoff) via `release` command.
- Integrated `search` and `release` into the unified Workspace CLI.
- Polished UI with tooltips and improved typography.

## [2.7.0] - 2026-05-20
### Added
- Scaffolded **Ecosystem Mood Stream** in AIOS (`moodRouter.ts`) for real-time visualization data.
- Implemented "INSANELY GREAT" branch consolidation (`--consolidate` flag in CLI).
- Created per-submodule `IDEAS.md` for all 10 core projects.
- Established persistent workspace memory in `src/workspace/core/memory.py`.
- Refactored entire management layer into `src.workspace` sub-packages.
- Transitioned project into Phase 4: Convergence.
